Question-2: What is the difference between innerHTML, innerText, and textContent ?

In JavaScript, "innerHTML" retrieves the complete HTML content within an element, including tags, while "innerText" only retrieves the visible text content of an element (ignoring HTML tags), and "textContent" retrieves all text content within an element, including hidden text and whitespace, but also ignoring HTML tags;

Question-3: What is event delegation in the DOM?

In the DOM, "event delegation" is a JavaScript technique where you attach a single event listener to a parent element instead of adding listeners to multiple child elements

Question-4: What is event bubbling in the DOM?

Event bubbling in JavaScript is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, use document.createElement() to create elements. Add them with appendChild(), append(), prepend(), or insertBefore(). Remove elements using remove() or removeChild(). Replace elements with replaceChild().
