Question-3: What is event delegation in the DOM?

Instead of adding an event listener to each child element, we add one listener to the parent and let it handle events for all its children.Here we use only one event listener.Event delegation use when we handling clicks on lists, tables, or menus and here elements are added dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts at the target element and then moves up (bubbles) through its parent elements until it reaches the document.We use event.stopPropagation() to stop event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

To create a new element, use the document.createElement() method.Once an element is created, we can add it to the page using different methods: 1. appendChild() - Adds at the end of a parent 2.innerHTML - Inserts elements as HTML 3.append() - Adds multiple elements or text.Removing Elements from the DOM- 1.remove() -Removes an element directly 2.removeChild() - Removes a specific child element
